## 5. 综合结论

| 实验 | 论文 L/S | 本 Run L/S | 论文 t | 本 Run t | 复现评价 |
|------|---------|-----------|-------|---------|---------|
| **Table 3 主实验** | 1.08%/月 | **1.08%/月** | 5.51 | **5.12** | **高度复现** |
| **Table 6 Top500** | 0.72%/月 | 0.53%/月 | 4.37 | 2.90 | **部分复现** |

### 关键发现

1. **训练设定对齐后，Table 3 主实验几乎完美复现**——L/S 均值与论文相同，P1/P10 端点与论文高度一致。
2. **Table 6 仍是主要差距**：L/S 偏低约 0.19%/月，P2 与 P10 拖累明显。
3. **Ensemble=5** 是唯一尚未对齐的训练超参；增至 30 可能进一步改善 Top500 及 t 统计量。
4. 本 Run 为**完整训练**（无 partial run 恢复），`meta.json` 含 `paper_sample_filter: true`、`feature_preprocessing: "none"`。
