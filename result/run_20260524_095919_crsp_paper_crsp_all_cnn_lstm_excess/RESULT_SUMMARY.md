| 实验 | 论文 L/S | 本 Run L/S | 论文 t | 本 Run t | 复现评价 |
|------|---------|-----------|-------|---------|---------|

| **Table 6 Top500** | 0.72%/月 | 0.51%/月 | 4.37 | 2.83 | **部分复现** |

### 关键发现

1. **训练设定已与论文完全对齐**（含 ensemble=30），Table 3 L/S **1.07%** 与论文 **1.08%** 几乎相同。
2. **Table 6 仍是主要差距**：L/S 低 0.21%/月；P2 崩塌、P10 偏弱是主要形态差异。
3. **ensemble 5 → 30 对 Table 3 几乎无影响**（1.08% vs 1.07%），**Top500 亦未改善**（0.53% vs 0.51%）——差距更可能来自回测实现细节或 CRSP 字段映射，而非 ensemble 数量。
4. 本 Run 为**完整训练**（22.3 h，无 partial run 恢复）。
